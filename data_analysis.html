Data Analysis

A personal project where I use Power BI and Tableau to display data about
netflix orifginal films and ....

For this project I used Power BI to produce a dashboard that displays data about netflix original movies. The data for the dashboard was gathered from online sources (Wikipedia & Kaggle) and was then transformed using Power Query Editor. The created tables were then joined and/or merged. I created measures in order to filter out the data and edited the vizualisation fields to make the dashboard look better.

For the second part of this project I created a dashboard using Tableau.
